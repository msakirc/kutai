# Write Review
Allow users to submit detailed reviews with star ratings and optional media.

## Header
- Page title "Write Review"
- Cancel button
- Business/Product name display

## Star Rating Selector
- 5-star rating interface with tap selection
- Rating labels (Poor, Fair, Good, Very Good, Excellent)

## Review Text Field
- Multi-line text input with placeholder "Share your experience..."
- Character count indicator
- Rich text formatting options (optional: bold, italic)

## Media Upload
- Photo/video upload button
- Uploaded media preview grid with remove option
- Maximum upload limit indicator

## Privacy Notice
- Brief privacy statement
- Username visibility reminder

## Action Buttons
- "Submit Review" primary button
- "Save as Draft" secondary button

## Bottom Sheet Helper
- Review writing tips (expandable)
- Community guidelines reminder
- Examples of helpful reviews
