# Wix ADI / Wix Studio AI / Wix Harmony

**Date:** 2026-05-09
**Confidence:** M
**Category:** Established (since 2016) AI site builder, evolved through three generations

---

## 1. What it is

Wix originally launched **ADI (Artificial Design Intelligence)** in 2016 — one of the earliest "answer questions, get a website" tools. By 2025-2026 the lineage has split:

- **Wix ADI** — legacy consumer-grade questionnaire-driven site generator, still present but no longer the headline.
- **Wix Studio AI** — designer/developer-grade AI capabilities inside the new pro-Wix product (Wix Studio), targeting agencies.
- **Wix Harmony** — announced as the next-gen successor, positioned as the evolution beyond ADI; details still emerging.

15+ AI tools across the suite cover layouts, text, images, branding, sitemaps, wireframes, image enhancement.

## 2. Input flow

- ADI: short questionnaire (industry, name, features needed).
- Studio AI: prompt-based generation inside the Studio designer; visual sitemap and wireframe tools generate from text.
- AI Image Creator: descriptive prompt → on-brand image inserted into site.

## 3. Output type

- Hosted Wix site, fully editable in Wix's editors.
- Visual sitemaps + wireframes (Studio).
- AI-generated images, text, schemas.

## 4. Iteration loop

Hybrid: AI scaffolds, designer/owner edits visually. Wix's positioning emphasizes "choose when and how to use AI" — opt-in per task, not always-on.

## 5. Spec / charter

None as a separate artifact. The visual sitemap step in Wix Studio is the closest thing to a structural pre-design / spec, but it's not exported as a doc.

## 6. Multi-screen / multi-page

Yes; Studio's sitemap tool makes multi-page architecture explicit.

## 7. Style / theming

Branding kit generated from prompt (colors, fonts). Style applied across all pages. Studio offers more granular control than ADI.

## 8. Deploy / export

Wix-hosted (primary). Wix has historically been the strongest example of platform lock-in — code export is limited.

## 9. Pricing

Site plans from ~$17/month (Light) up to ~$159/month (Business Elite); Studio has its own pricing for agencies.

## 10. Underlying model

Reported partnership with OpenAI; not all features attributed publicly.

## 11. Recent updates

- Wix Studio AI capabilities (sitemap, wireframe, image creator) — the agency push.
- Wix Harmony announcement — next-gen successor narrative.
- "Single-prompt site" capability matured.

## 12. Notable strengths / limitations

- **Strength:** Massive distribution + decade of iteration on the questionnaire-to-site flow. Cheaper than Webflow for non-technical owners.
- **Limitation:** Lock-in is severe; marketing-site bias; less developer-friendly than Webflow.
- **Lesson for KutAI:** Wix ADI's longevity (2016 → 2026) is proof that the "questionnaire → site" pattern survives, but it survives by getting denser (Studio's 15+ tools) rather than by replacing the questionnaire with a single chat prompt. KutAI's Telegram surface should prefer a small, dense menu of "what should we generate next?" over an open-ended chat that founders fill with low-signal text.
