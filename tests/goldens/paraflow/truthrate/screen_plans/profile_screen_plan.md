# Profile
User profile displaying personal information, reputation metrics, and account settings.

## Header
- Username display (large, prominent)
- User score display (e.g., "Score: 1,250")
- Credibility level badge (e.g., "Expert Contributor")
- Edit profile button

## Reputation Stats
- Total reviews contributed
- Total facts reported
- Helpful votes received
- Fact verification rate
- Member since date

## Recent Activity
- Last 3-5 contributions (reviews or facts)
- Each showing: business name, contribution type, date, upvotes received

## Settings Section
- Account settings option
- Notification preferences option
- Privacy settings option
- Theme toggle (Light/Dark mode switch)
- Help & Support option
- Log out button

## TabBar
- Home
- Search
- My Activity
- Profile
