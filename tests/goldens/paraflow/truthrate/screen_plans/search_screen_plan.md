# Search
Search interface for finding businesses, products, and services.

## Search Bar
- Search input field with placeholder "Search anything..."
- Search icon
- Filter button (category, location filters)

## Search Suggestions
- Recently searched items
- Popular searches
- Category quick filters (Restaurants, Products, Services, Health, etc.)

## Search Results (when query entered)
- Results list showing:
  - Business/Product image
  - Name
  - Category tags
  - Overall star rating with review count
  - Top verified fact snippet (if available)
  - Distance/location (if applicable)

## Ad Space
- Native ad card inserted every 4-5 results
- Banner ad at bottom

## TabBar
- Home
- Search
- My Activity
- Profile
