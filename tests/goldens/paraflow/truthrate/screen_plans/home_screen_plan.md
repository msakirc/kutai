# Home
Allow users to discover popular businesses, trending reviews, and access personalized recommendations.

## Search Bar
- Search input field with placeholder "Search anything..."
- Search icon button

## Featured Content
- Trending businesses cards with star ratings
- Popular categories chips (Restaurants, Products, Services, Health, etc.)
- Recent high-reputation reviews feed

## Quick Actions
- Single prominent button: "Contribute"
- Subtext: "Share your review or report a fact"
- Opens universal contribution flow

## Personalized Recommendations
- Recommended businesses based on user activity
- Community highlights from high-reputation reviewers

## TabBar
- Home
- Search
- My Activity
- Profile
