# CPS SP4a — husam-inline migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace `general_beckman.enqueue(..., await_inline=True)` with `husam.run(spec)` in the four legitimate LLM-consumer callers (vision, brand_voice_lint, copy_compliance_review, yalayut synthesize), removing the blocking primitive and the two nested-`await_inline` sibling-deadlocks.

**Architecture:** `husam.run(task: dict) -> dict` does select→execute→map directly (no pump), returns the legacy response dict `{"content", ...}`, and **raises** (`ModelCallFailed`/`RuntimeError`) on failure instead of returning a failed `TaskResult`. Each caller already builds (or can cheaply build) a `raw_dispatch` `context.llm_call` spec — exactly what `husam.run` rehydrates. The migration swaps the call, reads `resp.get("content")`, and maps a raise onto each caller's existing failure fallback. This is the `mr_roboto.critic_gate.produce_verdict` precedent (SP3b Task 8).

**Tech Stack:** Python 3.10, pytest (`pytest-asyncio`), `unittest.mock.patch`, packages `husam`/`general_beckman`.

**Spec:** `docs/superpowers/specs/2026-06-05-cps-sp4a-design.md`

**The four tasks are independent (no shared state).** Each: write/adapt the test to patch `husam.run`, prove it fails on the old code, migrate, prove it passes, commit.

**Critical environment rules (every task):**
- Run pytest **one invocation at a time**, never two concurrently (shared SQLite DB deadlock crash-loops live KutAI).
- Always `timeout`-prefix pytest, add `-p no:cacheprovider`, use the worktree-absolute python `C:/Users/sakir/Dropbox/Workspaces/kutay/.claude/worktrees/cps-sp4a/.venv/Scripts/python.exe` if the worktree has its own venv, else the main `.venv`.
- `husam` must be importable: confirm `pip show husam` resolves; if not, `.venv\Scripts\pip install -e packages/husam`.
- `husam.run` is patched as the string target `"husam.run"` (callers do `import husam; await husam.run(spec)`).
